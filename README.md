# Donezo
A minimal, visual productivity tracker focused on getting things done.

Features:

- Add tasks freely (any subject, any difficulty)
- Complete tasks with **confetti animation**
- Track **points, badges, streaks**
- Interactive **Chart.js subject visualization**
- Responsive design with **neutral, professional palette**
- Perfect for **MLSA portfolio & GitHub showcase**

Made by Warisha - MLSA Portfolio Project
